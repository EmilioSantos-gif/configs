## Getting Started

- ln -s $HOME/configs/shell/.config/zsh/.zshrc $HOME/.zshrc
- ln -s $HOME/configs/env/.aliases $HOME/.aliases
- ln -s $HOME/configs/env/.env $HOME/.env
- ln -s $HOME/configs/env/.env_local $HOME/.env_local
